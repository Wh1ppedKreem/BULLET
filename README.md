# **Installation**

## **[Linux](https://wikipedia.org/wiki/Linux) Installation**
### Clone the repository: 
```
$ git clone https://github.com/Wh1ppedKreem/EZToolkit.git
```
### Change directory to "EZToolkit": 
```
$ cd EZToolkit
```
### Run EZToolkit as root: 
```
$ sudo python3 eztoolkit.py
```
